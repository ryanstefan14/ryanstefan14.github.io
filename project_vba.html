VBA Automation: Unit Converter

This project was part of a broader MIS coursework where I utilized Visual Basic for Applications (VBA) within Excel to solve specific computational problems. The primary goal was to automate the conversion process between units, specifically handling the logic to convert kilometers to miles.

VBA Source Code

Sub Problem1_Convert()
    ' Variable Declaration
    Dim inputValue As Double
    Dim convertedValue As Double
    Dim conversionFactor As Double
    
    ' Initialize Variables
    conversionFactor = 0.621371
    
    ' Get Input from User
    inputValue = InputBox("Enter value in Kilometers:")
    
    ' Perform Calculation
    If IsNumeric(inputValue) Then
        convertedValue = inputValue * conversionFactor
        
        ' Output Result
        MsgBox "Result: " & Round(convertedValue, 2) & " Miles"
    Else
        MsgBox "Please enter a valid number."
    End If
End Sub
