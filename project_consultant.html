Project Overview

This is a Final Project I completed for Intro To Python Programming Class. This Python application supports ABC Consulting, Inc. in their annual employee performance analysis.

Key Features

  • Utilization Rate: Calculates hours worked vs. the 2,250-hour benchmark.
  • Sentiment Analysis: Scans evaluation text for positive (e.g., "excellent") vs. negative (e.g., "late") keywords.
  • Pay Raise Logic: Auto-calculates raises for consultants with >75% utilization and positive evaluation scores.
  • Data persistence: Reads/writes to emp_end_yr.txt.
